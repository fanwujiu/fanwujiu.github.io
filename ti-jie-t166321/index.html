[题解]T166321 [SgOI]奶茶

题目链接

1 | 算法分析

阅读题面可知，本题实际上要求 y=ϕ(x)y = \phi(x)y=ϕ(x).

(

∵lcm(x,ai)=xai\because lcm (x,a_i) = xa_i∵lcm(x,ai​)=xai​

∴gcd(x,ai)=1\therefore gcd (x,a_i) = 1∴gcd(x,ai​)=1

∴y=ϕ(x)\therefore y = \phi (x)∴y=ϕ(x)

)

2 | 代码实现

根据欧拉函数的定义，即可得出代码.

C语言描述如下：

int phi (int n)
{
	int ans = n;
	int i;
	for (i = 2; i <= sqrt (n); ++i)
		if (n % i == 0)
		{
			ans -= ans / i;
			while (n % i == 0) n /= i;
		}
	if (n != 1) ans -= ans / n;
	return ans;
}

3 | 代码优化

我们可以看到，在for循环的第二个条件中，目前写为i <= sqrt (n).可能有些人会写成i * i <= n，但是，虽然本题数据很水，但数据一旦大了，i * i就可能溢出．

我们知道，对于任意正整数 iii ,其约数总是成对出现,即只需找到较小的约数,就不需要再判断较大的约数.(i <= n / i).

再利用0的逻辑属性,将== 0的部分替换为!加前面的部分.

完整代码的C语言描述如下:

int phi (int n)
{
	int ans = n;
	int i;
	for (i = 2; i <= n / i; ++i)
		if (!(n % i))
		{
			ans -= ans / i;
			while (!(n % i)) n /= i;
		}
	if (n != 1) ans -= ans / n;
	return ans;
}
